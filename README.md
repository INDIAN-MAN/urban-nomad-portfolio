# Urban Nomad — Photography Portfolio (GitHub Pages)

This is a lightweight static photography portfolio site you can host on **GitHub Pages**.

## Quick deploy (GitHub Pages)

1. Create a new GitHub repo (example: `urban-nomad-portfolio`)
2. Upload **everything in this folder** to the repo root (keep the same structure)
3. Go to **Settings → Pages**
4. Source: **Deploy from a branch**
5. Branch: **main** and Folder: **/(root)**
6. Save — your site will be live at `https://<username>.github.io/<repo>/`

## Customize

- Edit `assets/site-data.json`:
  - brand name, tagline
  - contact (email / instagram)
  - photo titles + categories
- Replace photos:
  - Full images: `assets/images/*`
  - Thumbnails: `assets/thumbs/*`

## Notes

- No build tools, no frameworks — works as plain HTML/CSS/JS.
